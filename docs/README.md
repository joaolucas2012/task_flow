# 📸 Screenshots

Esta pasta contém as capturas de tela do projeto TaskFlow.

## Imagens Necessárias

Para que as imagens apareçam corretamente no README principal, adicione os seguintes arquivos nesta pasta:

### Tema Escuro
- `home-dark.png` - Página inicial em tema escuro
- `tasks-dark.png` - Página de tarefas em tema escuro
- `about-dark.png` - Página sobre em tema escuro

### Tema Claro
- `home-light.png` - Página inicial em tema claro
- `tasks-light.png` - Página de tarefas em tema claro
- `about-light.png` - Página sobre em tema claro

## Como Capturar

1. Execute o projeto: `npm run dev`
2. Navegue pelas páginas
3. Alterne entre tema claro e escuro
4. Capture as telas usando a ferramenta de screenshot do seu sistema

## Formato Recomendado

- **Formato**: PNG ou JPG
- **Resolução**: Mínimo 1280x720px
- **Qualidade**: Alta qualidade para melhor visualização
